# Translation QC (examples.en)

## Problem

`examples.en` (the English gloss for each `jp` sentence) is LLM-generated by
the same `build_prompt`/`compute_row` step in `build_db.py` that produces
`jp`/`jp_reading`/`mnemonic`. Some of these translations are wrong — not yet
characterized (no sample pulled, no error taxonomy built). Flagged for
follow-up, not investigated.
